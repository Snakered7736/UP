# Как запустить сайт ФК Анжил

## Шаг 1: Запустить backend

Открой командную строку (cmd) или PowerShell и выполни:

```bash
cd c:\up\backend
pip install -r requirements.txt
python seed_data.py
python app.py
```

Или просто запусти файл [c:\up\backend\start.bat](c:\up\backend\start.bat)

Backend запустится на `http://localhost:5000`

**Важно:** Не закрывай окно с backend, пока работаешь с сайтом!

## Шаг 2: Открыть frontend

Просто открой в браузере файл:
`c:\up\frontend\index.html`

Или любую другую страницу (shop.html, login.html, register.html)

## Шаг 3: Протестировать

1. **Регистрация:**
   - Открой [register.html](c:\up\frontend\register.html)
   - Введи email (или телефон) и пароль
   - Нажми "Регистрация"

2. **Вход:**
   - Открой [login.html](c:\up\frontend\login.html)
   - Введи те же данные
   - Нажми "Вход"

3. **Покупка:**
   - Открой [shop.html](c:\up\frontend\shop.html)
   - Нажми "КУПИТЬ" на товаре "Домашняя форма"
   - Кликни на иконку корзины справа вверху
   - Нажми "Перейти к оплате"
   - Заполни форму и нажми "Оформить заказ"

## Проверка что backend работает

Открой в браузере: http://localhost:5000

Должен вернуться JSON:
```json
{
  "message": "Football Club API",
  "version": "1.0"
}
```

## Если ошибка "Ошибка подключения к серверу"

Это значит backend не запущен. Проверь:

1. Backend запущен? (окно с `python app.py` должно быть открыто)
2. Нет ошибок в консоли backend?
3. Порт 5000 свободен? (если занят, останови другой процесс)

## Как посмотреть созданные заказы

После создания заказа, данные сохраняются в базе данных:
`c:\up\backend\football_club.db`

Можешь открыть её любым SQLite браузером (например, DB Browser for SQLite)
